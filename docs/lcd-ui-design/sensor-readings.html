DESIGN NOTES
• The three-column sensor grid uses LVGL FLEX/GRID layout: 3× LV_GRID_FR(1), gap 14 px, padding 22 px. Cards stretch to fill available height.
• "Stale" value colour stays COL_INK — the value is still the last known reading. Only the stripe, sparkline, and footer status text change to COL_WARN.
• "Error" value colour is COL_DIM (faded) — the displayed value is the last pre-error reading. Sparkline uses COL_DIM stroke, no fill area.
• Tab bar updated to 5 tabs (160 px each) to accommodate the new TREND tab (REQ-LD-000, SRS v1.2).
• Footer LVGL version string corrected from "LVGL 9.2" (v9 reference) to "LVGL v8.3.11".
