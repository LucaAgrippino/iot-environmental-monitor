DESIGN NOTES
• Fields map to lv_textarea (LV_USE_TEXTAREA=1); on tap, focus is set via lv_group_focus_obj() and the on-screen keyboard becomes visible.
• The numeric keyboard is a custom lv_btnmatrix (LV_USE_BTNMATRIX=1) rather than lv_keyboard; this gives full control over key labels and layout. Alternatively, the LVGL built-in lv_keyboard can be used with LV_KEYBOARD_MODE_NUMBER.
• APPLY button: implemented as lv_btn; disabled (COL_DIM) while any field is in error state; re-enabled once all validations pass (REQ-LD-140).
• Confirm modal (REQ-LD-150): full-screen backdrop at 60 % opacity (pre-blended on RGB565), modal 380 × auto, touch outside is blocked (no dismiss on backdrop tap).
• Config layout uses LVGL GRID: LV_GRID_FR(1) left + 280 px fixed right, gap 22 px.
• Validation rules checked inline (on each keypress): High > Low alarm, value within sensor physical range, poll interval 1–60 s. Error message displayed beneath the offending field.
