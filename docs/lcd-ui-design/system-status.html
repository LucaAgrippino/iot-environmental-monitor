DESIGN NOTES
• Status screen maps to lv_table (LV_USE_TABLE=1) or a flex-column of paired lv_label rows — no dedicated LVGL status widget exists.
• Degraded state: header pill changes to COL_WARN "⚠ DEGRADED"; an alert banner using COL_ERR / COL_ERR_TINT appears at the top of the content area.
• Affected cells in degraded state use COL_ERR_TINT (#21171A) background fill — pre-blended, no runtime alpha.
• REQ-LD-070 is intentionally sparse in the SRS; this screen implements the logical subset of: FW version, MCU, LVGL version, uptime, RTC time, per-sensor health (online/offline + last poll timestamp), active alarm count, and config state.
• "⚠ DEGRADED" pill animates to blink at 1 Hz (same pattern as ■ ALARM pill) when there are active alarms.
