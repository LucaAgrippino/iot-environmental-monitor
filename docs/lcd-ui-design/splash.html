DESIGN NOTES
• Splash is full-screen, no tab bar, no header bar, no footer bar.
• The progress bar animates from 0 % towards the current sub-step percentage; on splash exit it jumps to 100 % then the operational screen loads immediately.
• Minimum on-screen time: 1.5 s. Auto-exit: first valid sensor reading OR 5 s timeout (whichever first).
• LVGL v8.3.11 correction: the v9 reference incorrectly shows "LVGL 9.2" in the footer. Correct string: LVGL v8.3.11.
• Boot sub-steps (left to right progress): LCD driver init → SDRAM init → Config load → RTC init → Sensor bus init → First sensor poll.
• The splash dot in the top-right corner uses box-shadow: 0 0 8px COL_ACCENT to simulate a glow — implement in LVGL as a second concentric circle behind the dot.
