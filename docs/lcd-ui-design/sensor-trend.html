DESIGN NOTES
• Three panels stacked vertically in the content zone (104 … 444 px). Each panel uses LVGL FLEX row: 120 px label column + expanding chart area via LV_FLEX_GROW(1).
• Chart series implemented with lv_chart (LV_USE_CHART=1, confirmed in lv_conf.h). Use LV_CHART_TYPE_LINE; Y-axis auto-scales via lv_chart_set_range().
• Invalid entries (REQ-LD-190): call lv_chart_set_next_value(chart, series, LV_CHART_POINT_NONE) for each missing sample. LVGL's line-chart skips NONE points automatically — no interpolation across the gap.
• Sensor history buffer: 60 entries per channel (REQ-SA-185), 5 s polling (REQ-SA-180), 5-minute window (REQ-SA-190). Buffer is volatile RAM only (REQ-SA-200) — history is lost on power cycle.
• "ACCUMULATING" badge (state 1) shown while buffer is not yet full (entry count < 60). Badge disappears once 60/60 is reached and the rolling window is active.
• Channel colours: Temperature = COL_ACCENT (#7BAFD4), Humidity = COL_OK (#6FBF8E), Pressure = COL_WARN (#D4A84C). Area fill = 10 % tint of the series colour (pre-blended, no runtime alpha — RGB565 constraint).
• Tab bar TREND icon: ascending polyline <polyline points="3,17 8,11 12,14 17,7 21,4"/> with baseline <line x1="2" y1="20" x2="22" y2="20"/> — not in v9 icon set.
