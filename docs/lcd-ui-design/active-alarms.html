DESIGN NOTES
• Alarm list maps to lv_table (LV_USE_TABLE=1). The 5 px left stripe is a separate narrow lv_obj column inside each row with LV_GRID_FR(0) fixed to 5 px, coloured per severity.
• "■ ALARM" header pill blinks at 1 Hz / 50 % duty using lv_anim on the LV_OPA_COVER style property — identical to the pattern used on all alarm-state pills.
• Severity stripe colour: CRITICAL = COL_ERR (#CC6666), WARNING = COL_WARN (#D4A84C).
• REQ-LD-085 data shown per row: sensor name, alarm condition string, onset timestamp + elapsed time, severity chip.
• Empty state (REQ-LD-090): centred icon + "No active alarms" + subtitle. Header pill returns to "● RUNNING" (COL_OK).
• Footer counter "N ACTIVE · M SINCE BOOT" gives operators a quick persistent-count check without navigating away.
